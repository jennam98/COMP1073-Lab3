# COMP1073-Lab3

Lab 3


DESCRIPTION



Build a simple HTML/CSS/JavaScript interface that features an object constructor and an object based on a toy car product that you are provided on the website: ModelToyCars.com. You can choose any of the toy cars from the website.



INSTRUCTIONS



Complete each of the below tasks:

Carefully analyze every detail featured on the website for the toy car.
Create a local repository for the code that you are about to generate.
Develop a JavaScript constructor that would enable a programmer to create one or more instances of the product you have in front of you.
Include 10-15 object properties that might describe every aspect of the product - including various types of data.
Build a simple HTML/CSS page to create one or more object instances based on your constructor and feature the toy car product(s) in some imaginative way.
Perhaps add some interaction that would allow the user to access or modify the object properties somehow?
Stage, comment, commit, and push the code to a remote repository (to show your progress as you worked through the challenge). Missing detailed commits can lead to zero grades.
Have fun!


SUBMISSION
